# DEEP_LEARNING

A collection of Deep Learning concepts, experiments, and projects implemented using Python and popular machine learning libraries.

## Overview

This repository contains code, notebooks, and resources related to Deep Learning. It is designed for learning, experimentation, and building practical AI applications.

* Artificial Neural Networks (ANN)
* Deep Neural Networks (DNN)
* Forward & Backpropagation
* Activation Functions
* Loss Functions
* Optimizers
* TensorFlow & Keras
* Model Evaluation
* Data Preprocessing
* Regression Models
* Classification Models
* Model Deployment

## Installation

Clone the repository:

```bash
git clone https://github.com/Piyushayush21/DEEP_LEARNING.git
```

Move into the project directory:

```bash
cd DEEP_LEARNING
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run Jupyter Notebook:

```bash
jupyter notebook
```

Or execute Python scripts directly:

```bash
python filename.py
```
